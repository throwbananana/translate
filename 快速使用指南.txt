-------------------
1. 修复了Gemini API模型名称（从 gemini-1.5-flash 更新到 gemini-2.5-flash）

🚀 立即开始测试（3步完成）
------------------------

步骤1：安装依赖
  pip install -r requirements.txt

步骤2：运行程序
  python book_translator_gui.pyw

步骤3：测试翻译
  1. 程序启动后，会自动加载预配置的API Key
  2. 点击"浏览..."选择 sample_book.txt（示例文件）
  3. 点击"开始翻译"
  4. 观看进度条，等待翻译完成（约30-60秒）
  5. 翻译完成后，点击"导出译文"
  6. 选择保存位置，完成！


📋 关于"无法导出译文"问题的说明
--------------------------------

可能的原因和解决方法：

原因1：翻译还没完成
  ⚠️ 如果翻译正在进行中，译文内容还不完整
  ✅ 解决：等待进度条到达100%，看到"翻译完成！"提示

原因2：翻译失败
  ⚠️ 如果API调用失败，没有生成译文
  ✅ 解决：检查API Key是否正确，网络是否正常

原因3：模型名称错误
  ⚠️ 旧版本使用了 gemini-1.5-flash（已不存在）
  ✅ 解决：已修复，现在使用 gemini-2.5-flash

原因4：文件未加载
  ⚠️ 没有先加载要翻译的文件
  ✅ 解决：点击"浏览..."选择文件

现在这些问题都已经修复，并且添加了更清晰的提示信息！

💡 使用技巧
----------

1. 测试大文件前先用小文件测试
   - 使用提供的 sample_book.txt（约1400字符）
   - 翻译时间约30-60秒

2. 观察进度提示
   - 进度条显示翻译进度
   - 底部文字显示当前翻译第几段

3. 翻译过程中可以查看
   - 切换到"译文"标签页查看实时翻译结果
   - 即使未完成也能看到已翻译的部分

4. 可以随时停止
   - 点击"停止翻译"中止翻译
   - 已翻译的部分会保留，可以导出
