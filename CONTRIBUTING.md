# 贡献指南

## 基本约定

- `tests/` 仅放自动化测试
- `scripts/manual_tests/` 仅放手工验证脚本
- 不要提交运行时产物、本地数据库、备份文件或明文密钥

## 建议流程

1. 新建功能分支
2. 至少运行 `py -m pytest -q`
3. 如修改了手工脚本，确认 `run_manual_tests.bat` 对应入口可执行
4. 提交前检查文档是否需要同步更新
5. 不要把 `coverage.xml`、`pytest.log`、运行时日志或本地缓存一起提交

## 提交说明

- 提交信息建议使用简短祈使句
- UI 可见变更建议附截图
- API 相关变更请注明是否需要新的环境变量或配置项
