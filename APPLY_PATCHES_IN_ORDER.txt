按顺序应用（建议每一步都先检查再应用）

1) 基础修复
   git apply --check translate_repo_fixes.patch
   git apply translate_repo_fixes.patch
   git add -A
   git commit -m "docs/config/deps/model cleanup"

2) CI / pytest / OCR 诊断
   git apply --check translate_repo_ci_ocr.patch
   git apply translate_repo_ci_ocr.patch
   git add -A
   git commit -m "add ci pytest and ocr diagnostics"

3) release / translation memory 清理
   git apply --check translate_repo_release_cleanup.patch
   git apply translate_repo_release_cleanup.patch
   git add -A
   git commit -m "remove tracked tm db and add release docs"

如果某一步失败，可尝试：
   git apply --check --3way <patch>
   git apply --3way <patch>

如果仍失败，可尝试：
   git apply --check --reject <patch>
   git apply --reject <patch>

说明：
- git apply --check 只检查，不改文件
- git apply 会改工作区，但不会自动创建 commit
- --3way 与 --reject 不能一起使用
