Quick Start (3 Steps)
========================================

Step 1: Start the program
  Double-click: start.bat

Step 2: Select a file
  Click "Browse..." button
  Select: sample_book.txt (or your own file)

Step 3: Translate
  Click "Start Translation" button
  Wait for completion
  Click "Export Translation" to save

That's it!

Large files (> 10,000 chars)
  - Preview mode (first 10,000 chars)
  - Fast loading (< 1 sec)
  - Example: test_50k.txt

Very large files (> 100,000 chars)
  - Same preview mode
  - Still fast!
  - Example: test_500k.txt

Installation
========================================

Install dependencies:
  pip install -r requirements.txt

Required:
  - PyPDF2 (PDF support)
  - ebooklib (EPUB support)
  - beautifulsoup4 (EPUB parsing)
  - google-generativeai (Gemini API)
  - openai (OpenAI API)
  - requests (HTTP)

========================================
Troubleshooting
========================================

Problem: Batch file shows gibberish
Solution: Use start.bat (not the Chinese named files)

Problem: Python not found
Solution: Install Python 3.7+ and add to PATH

Problem: Dependencies missing
Solution: Run "pip install -r requirements.txt"

Problem: API error
Solution: Check API key in "Configure API" button
